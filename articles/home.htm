User Flow Description

After logging in or signing up, User gets to the Home screen. Here is access to all of the JobDocs features and functional nodes. Below is description of the main User Interface blocks and elements on the Home screen groupped by their functionality.

Adding documents

Tap on the button calls a shortcut menu, which provides choice: to take a photo of a document or add scanned documents from the library. JobDocs will lead User through several steps to store, configure and use documents.

For more information see the Working With Documents article.

Categories

The Categories block contains all the used categories (preselected and selected by User), number of documents in each category, calculated Contact hours (CE), etc.

Tap on a category name in the Categories list leads to the category page, where User can see files stored in the category, configure the category, access file options, share files, etc.

There is a ... button next to each category. Tap on the button calls a shortcut menu with available category actions, such as:

  • Rename
  • Share
  • Delete

For more information see the Categories article.

Alerts

The button displays the number of notifications. Tap it to see the list of notifications about upcoming document expiration.

For more information see the Alerts article.

Search

You can search documents stored in JobDocs. For this, tap the magnifier button at the top of the screen. Type in the name of the document or a category. The list of search results will contain found documents and provide the ... button next to each result. Tap on this button will call a shortcut menu with available file actions.

Sharing

You can share documents to e-mail, Dropbox, Google Drive, export to photoroll, save to a cloud, or print. This can be done by tapping the button. The list of sharable folders (categories) will appear. Select files, tap SHARE and select the target destination - sharing done.

For more information see the Sharing article.

Resources

There a lot of supportive literature, forums, blogs, etc. on the Web nowadays. Tap the button to get the links to those resources relevant to your job.

For more information see the Resources article.

Settings

Tap the button to

  • edit your Profile
  • setup Notifications (For more information see the Notifications article.)
  • get Help
  • read Privacy Policy
  • provide feedback
  • get information about the JobDocs application
  • log out

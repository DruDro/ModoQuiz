So, first of all, if you want to challenge friends, you need invite them to Modo Quiz or find your friends here, if they are already playing the game.

Invite friends from Facebook

Please follow the steps below to add friends from Facebook:

  • Tap the menu button

  • Tap FRIENDS

  • Tap INVITE

  • Search for a friend to invite

  • Tap INVITE FRIENDS

Add friends

If you know the username of a person in Modo Quiz, you can find that person and add as a friend. For this:

  • navigate to the FRIENDS screen as described above

  • enter the USERNAME at the bottom of the screen

    If such a user exists, he/she will be displayed in the list above.

  • Then tap ADD FRIEND

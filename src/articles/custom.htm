Log in to Modo Quiz Admin panel using your Modo account to gain ability to create custom topics and custom questions.

When logged in, you will see the Quiz page, where you can add custom topics and questions. Tap Create new

Here is the question creation page.

To create a new question, do the following:

  • Enter the question

  • Specify a custom topic. If such topic already exists, the question will be added to the topic, if there is no such topic - the specified topic will be created and the question added to it.

  • Specify 3 alternative answers

  • Specify the correct answer

  • Provide a hint

  • Tap SAVE

The newly created questions and topics will appear on the main page. You can edit and delete questions.

Create as many questions in a topic as possible. So questions in your quizs will not repeat from time to time, and they will be much more interesting to challenge.

If you provide less than 6 questions for a quiz, Modo Quiz will add the rest of needed questions from preset categories.

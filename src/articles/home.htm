From the Home screen you can start playing, add friends, call the Modo Quiz menu, see active or pending quiz challenges, go shopping, etc.

After a challenge is created and played at-least once, it gets to the Home screen. You will see all the current challenges here:

To learn how to add friends and challenge them, see the Add friends article. Then have a look at the How to Play article to learn more about game modes and how to start a quiz.

In addition, as you can see at the top of the screen, there is a coin button and the number of Modo Coins you have. Tap on this button to get to the Modoshop. There you can buy some more Modo Coins. Modo Coins give you several features, which will make you undefeatable in any quiz: see the Modoshop article to learn more about achivements and rewards.

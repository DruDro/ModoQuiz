The latest improvements of the product since 2015 include:

Ability to manage Quiz

On the app UI we will see the Custom category as one more carousel item.

Then different topics of this user's questions can be presented to user as sub-categories (list) under Custom Quiz category. Questions of user, eventually, will be presented there under each sub-category.

Feature turn-on

User should be able to activate via a code / qr code.

[Back-end]

We should have ability to manage the Quiz on a separate web page only. On this page we will have one page for each new quiz added and then 1 page for questions per quiz.

If there is less than 6 custom questions added by user to the "Cutsom" category, we should add rest of the questions by the similar approach we use in the "Random Challenge". Example: If user added only 4 questions, the rest 2 questions will be taken by the app randomly from the existing questions database.

  1. Admins can manage both - questions from preset categories as well as customers' questions.
  2. Users/customers can manage only their own questions. End users just log into their own profile (web page), and Create Quiz > add questions.

If there are not users to play with, show a message

Random challenge: If there are no users to play with, have a message in footer after searching for opponent text: "No active opponent right now!"

Make the posting to a wall in Facebook able to tag only specific friends

Share to FB friend makes it visible only to a friend and not to "All Friends".

Change font-color of the question based on what color the background has

For different background colors of questions, we should make some logic which will change the color of question font to be more readable. So, that on Yellow or light-red, -orange, or any other lighter backgrounds we use the "black" color (check the exact color-code in the app) of the app. While on the darker backgrounds we should keep using the white color for the purpose.

Correct wordings of Push messages and add new if they are absent

  • Game start: XXX wants to challenge you in ModoQuiz right now (is sent right after user was added/invited to the game).
  • Your turn: It's your turn in ModoQuiz
  • Result: Get the result of your ModoQuiz game with XXX

Store a collection of tokens for push notfications to be sent to all devices

In order to notify ALL devices by push notification in case user used same account from different devices, we need to improve the tokens storage and keep a collection of them instead of a single one like it was before.

Add the "Progress wheel" animation when action takes time

In order to make UI obvious and prevent multiple taps while sending invitation to friends, the progress wheel was added over the semi-transparent background over the screen. Repeated touches are not applied until the first request completes.

Change the sounds playing on Positive and Negative answers

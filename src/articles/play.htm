Quiz Mode

From the home screen, if "Start New Quiz" is tapped, we'll go to this screen. The CHOOSE YOUR GAME screen has three options:

Challenge A Friend

One on one challenge. There are 3 rounds of the quiz. 6 questions in each round. When a round is complete, the player who initiated the challenge has to wait until the opponent completes the round.

See the Challenge a Friend article for more details.

Random Challenge

In this mode, the system will try to match you with another player who has also tapped 'random challenge'.

This is a one on one quiz.

Challenge Friends

Multiple friends can take the same challenge (either found through a text code on Facebook wall, or as an in-app challenge to a friend you already know).

In a challenge quiz, there is only one phase. This means that every user answers all the questions in one go. There is no sync time to wait for the other users to finish.

See the Challenge Friends article for more details.

Join a Challenge

Otherwize, if you tap JOIN CHALLENGE on the Home screen, you will get to the CHALLENGE screen.

This enables you to search for a challenge with an identifier as seen on a friend's Facebook wall. The identifiers should be short and memorable. For example "Challenge by Eirik PÅske 2015" where the text in italics is user defined upon

See the Join a Challenge article for more details.

Quiz Gameplay

So there are 2 types of quiz: one-on-one and challenge

The main difference is that in the one-on-one mode there are rounds by 6 questions, and in the challenge mode - you answer all questions in one go.

In any quiz mode, you can use "aids", such as:

  • 50/50 - When this option is selected, 50% of the wrong answers are removed

  • More time - When this option is selected, the countdown timer adds 15 seconds.

  • Hint - When this option is selected a text hint is given to the player. This hint is relevant to the question to make it easier to answer.

  • Skip - When this option is selected, the player skips the current question with no loss in score.

See the Modoshop article to learn how to make aids available in the quiz.
